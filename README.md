Martel Sans Long
================

Martel Sans Long is a libre font development project (Latin-script-only at this time). It is a special version of the <a href="https://github.com/typeoff/martel_sans">Martel Sans</a> font family developed for a Braunschweig/Leipzig-based book designer duo.
